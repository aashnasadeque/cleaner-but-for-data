# Data Instructions

Download the "Online Shoppers Purchasing Intention Dataset" CSV from UCI:
- Official page: https://archive.ics.uci.edu/ml/datasets/Online%2BShoppers%2BPurchasing%2BIntention%2BDataset
- Direct download page: https://archive-beta.ics.uci.edu/dataset/468/online%2Bshoppers%2Bpurchasing%2Bintention%2Bdataset/files

Place the raw CSV file in this folder and name it:

```
online_shoppers_intention.csv
```

The pipeline expects this exact filename at:
`data/online_shoppers_intention.csv`
